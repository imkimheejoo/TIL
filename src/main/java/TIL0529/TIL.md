### 추상타입과 유연함
- 추상화 : 데이터나 프로세스 등을 의미가 비슷한 개념이나 표현으로 정의하는 과정
- ex) 경기도는 3/14일에 모든사람이 도지사를 투표한다.
       서울특별시는 3/12일에 만 19세 이상만 대통령을 투표한다.
       부산광역시는 3/10일에 여자만  교육감을 투표한다.
 ->모두 투표를 한다 (하지만 각 도마다 방법이 다름) -> 투표하는 메서드를 공통으로 추상화(인터페이스) 가능!
